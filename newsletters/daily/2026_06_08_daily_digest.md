<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">fäm + DXBinteract's full May 2026 Dubai map — 10,281 deals AED 28.9B, 40% of volume sub-AED 1M and ASP holding +3% YoY — gives buyers a concrete bracket to position their search.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Khaleej Times publishes the fäm Properties × DXBinteract Dubai-wide May 2026 transaction print: 10,281 total sales worth AED 28.9B, broken into apartments AED 14.6B, villas AED 7.2B, plots AED 4.2B and commercial AED 2.9B. The price-distribution table shows sub-AED 1M at 39.82%, AED 1–2M at 31.02%, and AED 2–3M at 12.41% of monthly volume.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The citywide ASP holds at AED 1,650/sqft (+3% YoY) even as 40% of deals are sub-AED 1M — a price-band split that lets buyers compute competing-demand depth before viewing. Dubai South leads the area table (1,357 deals / AED 1.6B), followed by Wadi Al Safa 3 at 983 deals / AED 1.7B.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Brokers writing CMAs on JVC, Arjan and Business Bay studios should validate against the price band each unit actually sits in, not the citywide average. A luxury print at AED 145M (Palm Jumeirah Signature Villa) and three AED 100M+ apartment trades anchor the upper tail.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Listing-side brokers writing May-2026 CMAs, buy-side underwriters sizing competing demand on a target band, and analysts reconciling fäm/DXBinteract vs Cavendish Maxwell methodologies.

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Cavendish Maxwell Jan–May 2026: 66,900 sales / AED 196.2B as May rolls over to ~9,500 deals (-27% MoM, -46% YoY) with Eid clipping ~3,000 sales</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Cavendish Maxwell's Jan-May cut shows Dubai residential still big (66,900 sales / AED 196.2B) but May rolled over hard — Eid plus selectivity drives -27% MoM, -46% YoY.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Cavendish Maxwell's Director Ronan Arthur publishes the Jan–May 2026 Dubai residential print: 66,900 sales worth AED 196.2B, with off-plan still ~74% of activity. Year-on-year the value is down from AED 217.8B in Jan–May 2025.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
May alone collapses to ~9,500 sales worth AED 22B (vs 17,600 deals and AED 54.8B in May 2025), with the week-long Eid Al Adha break taking an estimated 3,000 sales off the tape. Off-plan and ready both cooled in tandem — the ready slowdown is "even more pronounced" since March.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
This is the cleanest authoritative read on the May 2026 slowdown, confirming Fortune / AGBI / Property Finder narratives. For sellers, the 'measured buyer' framing means listings will sit longer at peak ask; for buyers, the -27% MoM skid is a negotiating wedge.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Sellers timing list-vs-hold decisions; brokers triaging stale inventory; underwriters stress-testing H2 2026 demand; developers calibrating launch cadence against an Eid-normalised ~12,500 deal baseline.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">DHCM to phase-vacate parts of Remraam from 1 July 2026 for 16–20-month remediation — RERA-index rent comp + AED 7K–16K relocation allowance</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">DHCM is about to take a sizeable Dubailand low-rise rental cluster offline for ~18 months — the rent comp formula is generous on paper but service charges keep ticking.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Dubai Holding Community Management has told Remraam tenants that buildings will be evacuated in phases starting 1 July 2026 for a preventative remediation programme lasting 16–20 months. Residents return after works; the operator pays rent compensation plus a relocation lump sum.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Rent compensation is months-out × the average RERA Rental Index value for that unit type, plus a relocation allowance scaling from AED 7K (studio) to AED 16K (3BR). Service charges continue during works — roughly AED 1,200/month for a 1-BR on top of any outstanding mortgage.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
This is the first formalised landlord-paid relocation programme of its size in a Dubai master community, removing ~18 months of supply from a Dubailand catchment already tight on family rentals. Brokers can match displaced families into Mudon / Town Square / Dubailand Residence Complex.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Remraam owners weighing comp-vs-service-charge math; Dubailand brokers building displaced-tenant lists; investors scanning for post-remediation re-launch entries; legal advisors reading the RERA-index comp precedent.
</p>
