<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

</p>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

Al Barari rent comps just stepped up materially — landlords with Collection / Reserve / Nests inventory should re-anchor renewals immediately. The 12.8% gross yield on AED 54M also flags how cash-flow-attractive trophy villas can be when matched to the right tenant.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">AED 367.3M Emirates NBD loan locks in a Czech-listed institutional landlord's 19 ultra-prime Dubai units for staged 2027+ resale across four branded clusters.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Emirates NBD Capital and Europe's largest commercial landlord CPI Property Group closed an AED 367.3M (~EUR 86M) facility on 21 May, secured against 19 ultra-prime Dubai residences across Bvlgari The Lighthouse (Jumeirah Bay), Casa Canal and One Canal (Dubai Water Canal) and Mr. C Residences Downtown — 15 still under construction, 4 completed.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>

Net effect: 19 near-ready ultra-prime units enter the secondary market over the next 24–36 months with a financed, motivated seller.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
One of the largest publicly-disclosed institutional debt facilities against a Dubai branded-residence portfolio — confirms ultra-prime stock remains bankable collateral at a tier-1 UAE lender while a European institutional seller adds 19 verified secondary units to the Jumeirah Bay / Dubai Water Canal / Downtown corridor.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Bulk-deal brokers wanting early access to CPIPG's 2027 divestment cadence, HNW buyers willing to bid against an institutional seller in a thin super-prime market, and lenders sizing branded-residence collateral coverage.
