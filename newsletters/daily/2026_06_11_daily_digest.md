<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">AED 560M / 80,000+ sqft Naia Island plot is now the largest single-plot UAE coastal land trade on record — and it cleared off-market.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
An 80,000+ sqft beachfront parcel on Shamal Holding's Naia Island (off the Jumeirah coast) traded off-market at AED 560M (~USD 152.5M, ~AED 7,000/sqft) to an unidentified European buyer via Dubai Sotheby's. Naia Island was launched in August 2025 as an off-market-only ultra-prime enclave.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The deal nearly doubles the prior May 2026 Naia Island record (53,000 sqft / AED 377M) and ranks among the largest single-plot UAE land trades ever. Sotheby's cites a buyer pool that is 45% Europe/UK — the European buyer fits the dominant pool.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
This print resets the UAE single-plot land benchmark and confirms that ultra-prime beachfront is still clearing with institutional HNW buyers even as the broader market cools. For owners of comparable coastal plots (Palm Jumeirah, Jumeirah Bay), the AED 7,000/sqft Naia comp is a fresh upper-bound reference.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Super-prime landowners benchmarking coastal plot values, ultra-HNW buyers tracking scarce beachfront inventory, luxury brokerages validating the off-market discretion model.

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">UAE FTA expands VAT-refund scope for Emirati homebuilders — pools, landscaping, smart-home systems, home gyms and full demolish-and-rebuild now eligible; ~AED 25K average saving, AED 1B+ refund book in 2026</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">FTA blesses pools, landscaping, smart-home gear and full demolish-and-rebuild as VAT-refundable for Emirati homebuilders — ~AED 25K per claim, AED 1B+ refund book this year.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
The Federal Tax Authority expanded the eligible-expense list for the UAE-nationals new-home VAT refund to now cover staff quarters, home gyms, smart-home and security systems, electronic doors, swimming pools, landscaping, and complete demolish-and-rebuild — all effective 1 January 2026.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
FTA quantifies additional annual savings at AED 200M (~AED 25K average per claim) and now projects total 2026 approved refunds above AED 1B vs ~AED 754M in 2025. Claims are routed through the updated FTA digital VAT-refund platform; awareness sessions are rolling out at district councils.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Pools, landscaping and smart-home categories were the largest 'unrecovered VAT' line items in Emirati self-builds — that gap is now closed. For villa-plot developers in Al Barsha South, Nad Al Sheba and Mirdif, this is a downstream demand tailwind that partially offsets the post-March cooling narrative.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Emirati self-builders and family architects, Dubai villa-plot developers, landscaping and pool contractors, smart-home integrators, tax advisers, and brokers in Al Barsha South / Nad Al Sheba / Mirdif / Al Khawaneej.
</p>
