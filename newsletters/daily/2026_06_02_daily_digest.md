<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Dubai stopped trading as one market this spring — Josh Corder maps which postcodes are speculative bubble and which are end-user demand.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>

Betterhomes CEO Louis Harding calls it Dubai's first genuine two- or three-tier marketplace.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Speculative-bubble shape: Marina (+113% off-plan / -46% ready), Business Bay (+80% / -6%), Palm Jumeirah (+12% / -2.5%). End-user-led: Greens (-9.7% off-plan / +18% ready), JVC (-1% / +10%), Creek Harbour (-20% / +5.7%). Outlier: Dubai South ready -53% YoY — the steepest drop in the breakdown.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>

</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Off-plan reservation buyers (especially Marina/Business Bay/Dubai South), assignment-resale brokers, mortgage lenders sizing LTV on ready vs off-plan, and end-users shopping Greens, JVC, Downtown, or Creek Harbour.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">UAE Civil Code rewritten: Federal Decree-Law 25/2025 — 12-month latent-defect window, mandatory usufructuary registration, statutory pre-contract disclosure duty</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">The 1985 Civil Code retires on 1 June 2026 — Dubai SPAs, usufruct deals and broker disclosure standards all shift with the new law.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Federal Decree-Law No. 25 of 2025 — the first full rewrite of the UAE Civil Code in 40 years — took effect 1 June 2026. Three reforms matter most: latent-defect liability doubles from 6 to 12 months from delivery, usufructuary-rights contracts must be registered or are void, and a new statutory pre-contract disclosure duty makes withholding material facts a civil liability.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Every Dubai SPA template, RERA Form A/B/F and off-plan reservation form drafted post-1-June needs a fresh review. Pre-1-June contracts stay under the old 1985 code; contracts signed after 1 June that are silent on governing law default to the new code.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Buyers gain a longer latent-defect runway for 2025/2026 off-plan handovers; brokers face a documented disclosure duty that turns 'I didn't know' into litigation risk; and operators with usufructuary positions in DIFC, ADGM or JAFZA need to confirm registration before the deadline or risk nullity.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Dubai conveyancers and SPA drafters, broker compliance leads at RERA-licensed agencies, free-zone operators on usufructuary positions, and off-plan buyers receiving 2026 handovers.
</p>
