<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">MAF invokes force-majeure on Tilal Al Ghaf SPAs — Iran-war supply disruption cited, buyer delay-compensation clock at risk</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Tilal Al Ghaf force-majeure notice flips the off-plan buyer-protection switch — and re-prices delay risk across every comparable Dubai SPA.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Semafor reports Majid Al Futtaim has begun sending 'some customers' at Tilal Al Ghaf force-majeure notifications under their SPAs, citing post-Feb 28 Iran-war supply-chain disruption. The masterplan mixes &gt;USD 15M mansions with investor-grade units across 3,500 homes.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Standard Dubai off-plan SPAs entitle buyers to delay compensation if handover slips 6–12 months — invoking force majeure can pause that clock. Stonehaven estimates Gulf building-material costs are up ~25% since Sep 2025 (aluminium, bitumen, copper, nickel).
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Every Dubai off-plan buyer underwriting a 2026–27 handover must now re-read the force-majeure clause in their SPA. Sellers of partially-paid contracts should expect harder questions from secondary buyers about Oqood status and remaining payment exposure.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Tilal Al Ghaf off-plan buyers, secondary-market investors holding any Dubai 2026–27 SPA, brokers handling assignments, and lawyers structuring delay-compensation claims.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">MAF publicly denies Tilal Al Ghaf delays — buyer notices reframed as standard SPA disclosures, not a delivery-timeline change</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">MAF publicly rejects the Semafor force-majeure story 24 hours after publication — buyer notices reframed as standard SPA disclosures, not a delivery-timeline change.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
On 12 June Majid Al Futtaim issued a public statement denying it had invoked force-majeure or delayed homes at Tilal Al Ghaf (3,500 homes including 130-villa Alaya phase and 94-unit Elysian Mansions). The denial directly contradicts the 11 June Semafor report.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
MAF says construction 'continues to progress in line with the established project schedule and the terms of the SPAs'. The buyer notices Semafor surfaced are recharacterised as 'standard procedural notifications issued under the SPA framework'. No specific handover dates were re-confirmed and input-cost claims were not directly addressed.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
For resale negotiations, the developer's public on-schedule affirmation is useful context. The risk: MAF declined to publish updated milestone dates, so the actual cost-absorption question (who eats the ~25% input-cost jump) remains open.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Tilal Al Ghaf SPA-holders considering assignment exits, buyer-side brokers running force-majeure clauses against developer reps, and analysts modelling Dubai 2026/2027 handover slippage.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">World Bank cuts UAE 2026 GDP to 2.4% (from 5%) on Hormuz closure — GCC-wide down to 1.3%, 2027–28 rebound queued</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">The macro pillar under Dubai's 'fundamentals intact' frame just got revised down 2.6 percentage points for 2026.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
World Bank's June 2026 Global Economic Prospects revision cuts UAE real GDP for 2026 from 5% to 2.4%, with GCC slashed by 3.1pp to 1.3%. Recovery projections of 4.1% (UAE 2027) and 5.2% (GCC 2027) are queued as the Hormuz shock is expected to fade.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The downgrade is directly tied to the Iran-Israel war (Feb 28 start) and Strait of Hormuz closure, which choke oil routing and dent regional GDP. Alternative pipelines bypassing Hormuz are flagged as a UAE-specific credit-positive offset — explaining why the UAE's cut (-2.6pp) lands softer than the GCC average (-3.1pp).
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
A 2.6pp UAE GDP cut for the current year is the first hard macro mark against the 'fundamentals intact' narrative — the kind of number lenders, REIT analysts and family offices retest underwriting models against. The 2027–28 rebound numbers keep this framed as a one-year shock, not a regime change.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Portfolio allocators, REIT and developer credit analysts, mortgage-LTV and AED-buyer-currency desks, and macro-aware property advisors.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai May luxury off-plan: AED 4.96B (USD 1.36B) across 391 AED 5M+ deals — villas AED 2.51B edge apartments as HNW bids hold</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Luxury off-plan held its AED 5B pace through Dubai's softest May print in years — villas are doing the heavy lifting.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Keturah analysis of DXBinteract data covers every Dubai off-plan transaction above AED 5M in May 2026: AED 4.96B across 391 deals — roughly 12 luxury tickets per day — with average value AED 12.7M.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Villas booked AED 2.51B across 184 transactions; apartments AED 2.45B across 207. The villa AED 10–20M band led with 60 deals / AED 834.2M. On apartments, 76% of deals (158 of 207) sat in the AED 5–10M slab — first-rung branded-residence territory at projects like Keturah Reserve and Ritz-Carlton Residences.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
May 2026 was the broader market's softest run in years (Cavendish Maxwell: ~AED 22B, -27% MoM, -46% YoY). Yet luxury off-plan held ~AED 5B and the average ticket rose — confirming HNW capital still bids scarce branded inventory even as mid-market waits.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Luxury off-plan brokers, branded-residence developers, AED 5M+ buy-side advisors, and HNW relocation funds.
