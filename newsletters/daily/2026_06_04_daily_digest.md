<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">ADREC freezes Abu Dhabi rents to 0% — Dubai's 5% RERA cap is now the UAE's most landlord-friendly regime</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Abu Dhabi caps every rent at 0% indefinitely — residential, commercial, industrial — replacing the 5% cap in place since 2016.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
ADREC froze all Abu Dhabi rent increases to 0% effective 2026-06-02 — every residential, commercial, and industrial renewal and new lease must match the prior contract rate, indefinitely replacing the 5% annual cap in force since 2016.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Renewals and new contracts are processed at the existing rate with immediate effect; no landlord notice already served for an increase is now enforceable. Dubai's RERA calculator — permitting stepped increases up to 20% when rent is materially below the index — is untouched.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
AD landlords facing margin compression have fresh incentive to redeploy into Dubai rental stock. Dubai's 5% cap is now the UAE's most landlord-friendly rental regime by a wide margin — widening the effective yield premium between the two emirates.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Dubai landlords reunderwriting yield, Abu Dhabi-to-Dubai capital rotators, leasing brokers building cross-emirate pitch decks, and REIT analysts pricing UAE rental regimes.
</p>
