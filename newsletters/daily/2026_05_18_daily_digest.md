<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai 2026 handovers face 6–9 month delays — Fitch and Anarock flag ~22,500 units slipping into 2027+</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Fitch and Anarock both call it: the bulk of Dubai's promised 2026 keys land in 2027 — a supply pause that resets rental, off-plan and yield maths simultaneously.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Two independent May-12 reports converge: Fitch estimates 6–9 month delays across the 45,000-unit 2026 handover slate, while Anarock puts ~half (~22,500 units) sliding into 2027+. 58% of Dubai's 1,592 active projects sit at 0–20% completion — the most exposed cohort.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Strait of Hormuz reroutes, ~20% Alba capacity cut, ~12-month EGA Abu Dhabi remediation and ~30% input-cost inflation are driving the stall. Asian shipping detours add 7–15 extra days; ceramics, aluminium and MEP components are up 18–28%.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Check escrow draw schedules and developer force-majeure language on any unit due Q3–Q4 2026 — Dubai law allows up to 12-month delays without buyer rescission rights. Landlords renewing in late 2026 face less new-supply pressure than the 45k-unit headline implied.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Off-plan buyers due Q3–Q4 2026 keys; landlords negotiating 2026 renewals; analysts modelling Dubai rental supply through 2027; developers communicating handover slippage to escrow agents.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai realty regains momentum — DLD +2% MoM in April, tenant enquiries +40%, rental inventory doubles to ~2,200 units</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Eight weeks into the regional conflict, Dubai sales enquiries are +11%, tenant enquiries +40%, rental stock has doubled and ~70% of listings have already adjusted ask by ~10%.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Khaleej Times' 17 May read on combined Betterhomes and Bayut/Dubizzle Property data — the cleanest mid-May snapshot of how the Dubai market is absorbing the 28-Feb regional shock. DLD transactions rose ~2% MoM in April with sales and tenant intent recovering.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Betterhomes reports sales enquiries up 11% and tenant enquiries up 40% Mar→Apr, while rental inventory more than doubled from ~1,000 to ~2,200 units; ~70% of listings underwent price adjustments averaging ~10%. Bayut/Dubizzle records active users at 85% of 2026 baseline, unique buyers at 87%.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Underwrite Dubai rentals on a softer landlord stance: supply doubled and ~70% of listings cut ~10% within eight weeks, compressing rent-comp tables fast. Buy-side, the +11%/+40% enquiry curve argues buyers are returning — but with negotiation leverage.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Dubai landlords repricing rents, agency leasing teams chasing the 40% intent uplift, buy-side investors triangulating where to deploy, and analysts marking April DLD against March prints.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Etihad Credit Bureau opens tenant credit-score screening for UAE landlords via UAE PASS — plus AI cheque-clearance scoring</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">AECB + UAE PASS now lets Dubai landlords pull a prospective tenant's credit score (and an AI cheque-clearance score) before signing — the first national pre-lease diligence rail.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Etihad Credit Bureau, with TDRA and Digital Dubai, has switched on a new tenant-screening flow. Landlords request a prospective tenant's credit score via the AECB app or web portal; the tenant authorises the data share via UAE PASS, after which the landlord sees the score plus an AI-driven cheque-clearance likelihood.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Landlord initiates request in AECB → tenant receives a consent prompt in UAE PASS and authorises → AECB returns the credit score plus an AI cheque-clearance score predicting the probability of post-dated rent cheques bouncing. Verify fees and effective date directly with AECB before relying on it for a live tenancy decision.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Until now, Dubai landlords had no national, consent-based tenant credit channel — diligence ran on payslips, NOCs and broker grapevine. A live AECB rail re-prices risk for tenanted-yield deals and gives brokers a new value-add at lease signing.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Dubai landlords screening prospective tenants, leasing brokers building higher-trust tenancy stacks, build-to-rent operators with portfolios of post-dated cheques, and yield-investors underwriting tenanted purchases.
</p>

The four rated developers — Emaar, Damac, Binghatti, Arada — sit on low-to-moderate leverage with front-loaded payment plans.

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Developers are expected to defend asset values by widening payment plans rather than cutting headline prices. Limited capital-market dependence means no forced refinancing over the next 12 months — downside risks are prolonged Hormuz disruption and sustained collection slowdowns pressuring liquidity through 2027.
