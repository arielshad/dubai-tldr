<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">fäm Properties + DLD: 69.9% of Dubai primary buyers since 2012 still own — 740,219 properties never resold, retention now matches London/NYC</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Three-quarters of Dubai's post-2012 buyers never sold — fäm + DLD say the flip cycle is gone and retention now matches London/NYC.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
fäm Properties analysed 1.1M+ DLD transactions over 16 years and found 740,219 homes have never been resold — 69.9% of all primary-market buyers and 61.1% of resale buyers still own the unit they bought.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Cohort analysis of 687,406 primary (2012–2025) and 425,083 resale (2009–2025) transactions shows a steepening retention curve: 42% of 2014 buyers, 53% of 2017 buyers and 61% of 2022 buyers still hold — attributed to the 2019/2022 Golden Visa, off-plan protections, infrastructure, and maturing family-led demand.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
The resale pipeline is thinner than volumes suggest — most owners aren't sellers, so supply skews to motivated/forced rather than discretionary. Dubai's hold profile now structurally mirrors London and New York.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Brokers building seller pitches, off-plan operators sizing post-handover supply risk, ready-market underwriters benchmarking liquidity, and analysts updating Dubai vs global hold-period models.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">First time in years that valuations professionals are openly calling Dubai/UAE a buyer's market — villas confirmed printing at-or-below fair value.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
The National's 08-May piece is the first major mainstream call that the UAE has shifted into a buyer's market — CBRE Mena Head of Valuations, BlackBrick CEO Matthew Bate (villas clearing "at fair market value and in some cases slightly below"), and S&P (AA/A-1+ reaffirmed) all align in the same week.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Q1 sales-price growth slowed to ~9% YoY (off the post-pandemic double-digit run-rate), March transactions fell ~20% on regional-conflict shock, and named villa communities — Victory Heights, Jumeirah Golf Estates, Masakin Al Furjan — are clearing at or below fair value.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
When a Big-3 valuations house, an established prime brokerage, and S&P align simultaneously, ask-vs-fair-value spreads have materially compressed — sellers holding out for 2025 peak prints are now explicitly in the minority view.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Villa buyers in mid-prime estates, sellers re-pricing 2025 carrying assumptions, valuations and underwriting teams, family-office allocators needing a long-horizon thesis.

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Four straight sessions of off-plan share collapsing from 78% to 53% — 08-May confirms the rotation toward ready stock is real.</p>

<strong style="color: #f7ff00;">What is it?</strong><br>
Dubai's 08-May DLD print landed at AED 1.14B with ready taking 47.3% (AED 539.2M) versus off-plan's 52.7% (AED 600.7M) — the lowest off-plan share of any May 2026 session and the fourth consecutive session of compression from the 78.4% high on 05-May.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Off-plan share trajectory: 05-May 78.4% → 06-May 71.2% → 07-May 61.8% → 08-May 52.7%. Ready villas alone cleared AED 151.6M (28.1% of ready) — a clean end-user demand signal that is not single-tower noise.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>

</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai Municipality readies shared-housing permit portal under Law 4/2026: 12-month grace for existing operators, fines up to AED 1M for repeat violators</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Dubai's shared-housing permit portal is about to go live — partitioned-unit operators have a 12-month grace window to legalise or face escalating fines.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Dubai Municipality confirmed the digital permit portal under Law No. (4) of 2026 is being finalised — existing partitioned-unit operators get a 12-month grace window once it opens, with procedures and fees to be announced shortly.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Only owners or licensed operators can run shared units (tenants can no longer sublease partitions). Permits are 1-year renewable; fines run AED 500–AED 500,000 per violation, doubling on repeat within a year up to a maximum of AED 1M.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Crackdowns are already active in Al Rigga, Al Muraqqabat, Al Barsha, Al Satwa and Al Raffa — once the portal opens, the grey-zone niche becomes either fully licensed or fully penalised, reshaping yield calculations for the sub-AED 1M studio segment.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Landlords leasing partitioned units, co-living and bedspace operators, and investors underwriting sub-AED 1M studios in Deira, Bur Dubai and Al Barsha.
</p>
