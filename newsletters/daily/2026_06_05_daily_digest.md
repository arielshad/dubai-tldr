<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai Holding to Phase-Vacate Remraam from July 1 for 16–20 Month Remediation</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">DHCM is taking a sizeable Dubailand low-rise rental cluster offline for ~18 months — rent comp formula is generous on paper but service charges keep ticking.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Dubai Holding Community Management has notified Remraam tenants that buildings will be evacuated in phases starting 1 July 2026 for a preventative remediation lasting 16–20 months. Rent compensation = months out × average RERA Rental Index for the unit type, plus a one-off relocation lump sum (AED 7K studio → AED 16K 3-BR).
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Service charges still run during works (≈AED 1,200/month for a 1-BR), so landlords must model the net hit even if displaced tenants are made whole on rent. Residents return after the remediation window ends.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
This is the first formalised landlord-paid relocation programme of its scale in a Dubai master community. It removes 16–20 months of supply from a Dubailand catchment already tight on family rentals, and sets a RERA-index comp precedent for future remediation events.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Remraam owners weighing comp-vs-service-charge math; Dubailand brokers building displaced-tenant lists; investors scanning for post-remediation re-launch entries; legal advisors reading the RERA-index comp precedent.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">ADREC Freezes Abu Dhabi Rents to 0% — Dubai's 5% RERA Cap Is Now the UAE's Most Landlord-Friendly Regime</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Abu Dhabi caps every rent at 0% — Dubai's 5% RERA cap is suddenly the UAE's most landlord-friendly rental regime.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
On 2026-06-02, ADREC suspended all rent increases across residential, commercial and industrial leases in Abu Dhabi — both renewals and new contracts on previously-rented units must match the prior contract's rate, effective immediately and until further notice. The 5% annual cap in place since 2016 is overridden.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Dubai's regime is untouched — the RERA Rental Increase Calculator still permits stepped increases up to 20% when existing rent is materially below the index. That widens the cross-emirate spread on what a landlord can underwrite per AED of capital deployed.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
AD landlords facing margin compression now have clear incentive to redeploy into Dubai rental stock. The longer the freeze holds without Dubai matching, the wider Dubai's effective yield premium becomes — a structural tailwind for Dubai buy-to-let underwriting.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Dubai landlords reunderwriting yield; AD-to-Dubai capital rotators; leasing brokers building tenant pitches; Dubai tenants assessing relocation risk; REIT analysts pricing UAE rental regimes.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
DHG Properties has confirmed construction started on Helvetia Marine — its third Dubai project and first on Dubai Islands. The 63-residence, 12,938 sqm BUA low-rise (1/2/3-BR apartments, duplexes with 5-metre ceilings, garden residences) is fully sold out from launch and targeting Q1 2028 handover. DHG's active UAE pipeline now totals AED 1.3B (~$350M) across three projects.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Assignment-resale supply is now the only obvious access route. Sister project Helvetia Residences in JVC hands over in July 2026, putting a real DHG rental-yield benchmark on the table within weeks — that print will be the leading indicator for Helvetia Marine's resale floor.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Dubai Islands has recorded ~AED 3.5B ($953M) of sales this year alone. DHG's sold-out launch rate confirms boutique waterfront product on the islands is clearing in days, cementing Dubai Islands as the 2027–2028 handover frontier for branded mid-luxury.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Dubai Islands assignment-resale hunters; brokerages prospecting Swiss/European buyers; rental underwriters modelling 2028 absorption; analysts tracking sold-out launch rates as cooling-market indicators.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">DLD Daily Print 2026-06-03: AED 1.73B — Ready Share Rebuilds to 44.8% Over Three Sessions</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Ready stock continues to rebuild share — 44.8% of yesterday's AED 1.73B DLD tape, the firmest ready print since the May Eid lull.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
DLD-registered transactions on 2026-06-03 totalled AED 1.73B across 10,218 deals: AED 957.2M off-plan (55.2%) vs AED 776.5M ready (44.8%). Off-plan flats led at AED 839.9M; ready flats cleared AED 520.8M and ready villas held AED 142M (18.3% of the ready segment).
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Ready share has now rebuilt three sessions in a row: 51.9% → 31.5% → 44.8%. Off-plan flat dominance (87.8% of the off-plan segment) also confirms the pipeline is still being pre-sold despite cooling headlines. Sample size of 10,218 deals is comparable to early-May daily runs before the regional shock.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
The post-Eid two-day average (~AED 1.1B/day) tracks the May weekly run despite holiday distortion — the cooling narrative needs to contend with daily volume still printing AED 1B+ per business day. For anyone modelling June DLD tape vs Fitch's -15% prediction, the June 3 print is on the resilient side.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Investors timing entries on ready stock; brokers underwriting Q3 listings; developers reading the post-Eid demand floor; analysts modelling the off-plan/ready rotation.
