<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Cavendish Maxwell Jan–May 2026: 66,900 sales / AED 196.2B — May collapses -46% YoY as Eid clips 3,000 deals</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Cavendish Maxwell's Jan-May cut shows Dubai residential still big but May rolled over hard — Eid plus selectivity drives -27% MoM, -46% YoY.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Cavendish Maxwell's Ronan Arthur has published the Jan-May 2026 Dubai residential print: 66,900 sales worth AED 196.2B, with off-plan still ~74% of activity and the year-on-year value gap vs Jan-May 2025 (AED 217.8B) running at -9.9%.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
May alone lands at ~9,500 deals / AED 22B — down -27% MoM and a stark -46% YoY vs May 2025's 17,600 deals — with Eid Al Adha accounting for ~3,000 of the volume gap and buyer selectivity doing the rest.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
This is the cleanest authoritative read on the May 2026 slowdown, confirming Fortune / AGBI / Property Finder narratives with a single verified dataset. The 'measured buyer' framing means listings will sit longer at peak ask and negotiation room has widened.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Sellers timing list-vs-hold decisions, brokers triaging stale inventory, and underwriters stress-testing H2 2026 demand against an Eid-normalised ~12,500-deal baseline.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">DHCM to phase-vacate Remraam from 1 July for 16–20-month remediation — RERA-indexed rent comp + AED 7K–16K relocation lump sum</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">DHCM is about to take a sizeable Dubailand low-rise rental cluster offline for ~18 months — the rent comp formula is generous on paper but service charges keep ticking.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Dubai Holding Community Management has notified Remraam tenants that buildings will be phased-vacated from 1 July 2026 for a preventative, long-term remediation programme lasting 16–20 months.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Rent compensation = months displaced × average RERA Rental Index value per unit type, plus a one-off AED 7K (studio) to AED 16K (3-BR) relocation allowance — though service charges (~AED 1,200/month for a 1-BR) continue running through the works period.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Dubai's first formalised large-scale landlord-paid relocation programme removes 16–20 months of Dubailand supply from a catchment already tight on family rentals, setting a RERA-index comp precedent with broader legal implications.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Remraam landlords running the comp-vs-service-charge net math, Dubailand brokers building displaced-tenant shortlists for Mudon/Town Square alternatives, and legal advisors tracking the RERA-index comp precedent.
</p>
