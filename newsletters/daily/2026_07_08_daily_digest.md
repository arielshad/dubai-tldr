<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Knight Frank H1 2026: Dubai $10M+ prints 296 deals for USD 5.1B — Aman Residences Jumeirah Second AED 422M apartment the top line</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Knight Frank Jul-7 H1 print: 296 $10M+ deals for USD 5.1B with Aman Residences Jumeirah Second's AED 422M six-bedroom apartment the top line.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Gulf News × Knight Frank's H1 2026 read: 296 deals cleared USD 5.1B in Dubai's $10M+ residential tape — up 14% YoY on value, 49% above H1 2024 on count, with a record 26 transactions above USD 25M. Top line: a six-bedroom apartment at Aman Residences (Jumeirah Second) cleared USD 114.9M / AED 422M.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Dubai Hills Estate (51 deals), Palm Jumeirah (50), and Palm Jebel Ali (40) take ~48% of the luxury tape. Knight Frank's Faisal Durrani flags most H1 deals closed before regional conflicts and registered on standard DLD processing delays.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Palm Jebel Ali's 40 pre-handover deals before its 2028 completion sets a price floor for when physical stock lands. Q2 softened to 131 deals vs. Q1's 165 — H2 prints will confirm whether momentum holds or the sub-$10M asking-cut trend extends upward.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Ultra-prime brokers pitching Dubai Hills / Palm Jumeirah / Palm Jebel Ali; funds tracking pre-handover paper; luxury investors underwriting Aman Residences comps.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai Loop moves from design to execution — Boring Company × RTA begin 45,000-tonne tunnel lining production for DIFC↔Dubai Mall Phase-1</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Dubai Loop just cleared its first execution milestone — precast concrete production has started on 45,000 tonnes of tunnel lining for the DIFC↔Dubai Mall Phase-1 underground link.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
The Boring Company and Dubai's RTA have started producing ~25,000 precast concrete tunnel segments (~45,000 tonnes) for Dubai Loop Phase 1 at TBC's Dubai facility — the first physical-execution milestone since the May-18 Parsons design lock.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Phase-1 covers 6.4 km and four stations — DIFC, ICD Brookfield Place, Za'abeel/Dubai Mall Parking, and Burj Khalifa/Dubai Mall — at AED 600M (~$163M) capex, compressing the DIFC↔Dubai Mall commute from 20 minutes to 3 minutes with a ~2-year build horizon.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Station catchments have historically driven 5-15% premium in Dubai. Precast-production start is a credibility milestone — the project is now much harder to cancel — which starts baking commute compression into 2027-2028 acquisition pricing.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
DIFC and Downtown office tenants and landlords pricing 2027-2029 lease renewals; Burj-Khalifa-cluster apartment sellers timing an exit; ICD Brookfield Place owners; hotel asset managers in the Address/Armani cluster.
</p>
