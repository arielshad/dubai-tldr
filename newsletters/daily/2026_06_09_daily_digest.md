<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Naia Island sets new UAE land benchmark — 80,000+ sqft beachfront plot trades for AED 560M via Dubai Sotheby's, nearly doubling the May 2026 record</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">AED 560M / 80,000+ sqft Naia Island plot is now the largest single-plot UAE coastal land trade on record — and it cleared off-market.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Dubai Sotheby's has closed the sale of an 80,000+ sqft beachfront parcel on Shamal Holding's Naia Island at AED 560M (~USD 152.5M) to an unidentified European buyer. Naia Island launched in August 2025 as a curated, off-market-only ultra-prime enclave — the trade was completed without public marketing.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Plot size is nearly double the prior Naia Island record (53,000 sqft / AED 377M in May 2026), with an implied ~AED 7,000/sqft — consistent with the May comp. Typical Jumeirah Bay Island plots run 14,000–25,000 sqft; this single trade is multiples of the comparable next-best inventory.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
This print resets the UAE single-plot land benchmark and confirms Naia Island as Dubai's reference ultra-prime land asset alongside Jumeirah Bay Island. For super-prime brokerages, it validates the off-market discretion model over portal-driven listing.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Ultra-prime brokers underwriting Naia/Jumeirah Bay/Palm plot listings, family-office land allocators benchmarking Dubai coastal AED/sqft against Riviera/Aspen comps, developers exploring boutique branded-residence plays on adjacent coastal land.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai's First-Time Home Buyer Programme scales to AED 5B+ across 3,200+ buyers — 9 new developers join, total hits 22 partners and ~45K registrations</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">DLD's First-Time Home Buyer Programme passes AED 5B / 3,200 owners and onboards 9 new developers — the renter-to-owner pipe is now a 22-developer marketplace.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
DLD and the Department of Economy and Tourism report the First-Time Home Buyer Programme (launched July 2025) has enabled 3,200+ Dubai residents to register a freehold home for the first time, with AED 5B+ in cumulative transactions and ~45,000 registrations. Nine new developers join (Arada, SAMANA, Reportage, DWTC, 4Direction, IRTH, Manam, Qube, Sky View), taking the total to 22.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Eligibility is all UAE residents 18+ without an existing freehold property in Dubai. Five participating banks supply structured first-time mortgage products, integrated with DLD's freehold registration rail and DET's developer onboarding.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
The programme is now the single largest formal demand-creation lever for sub-AED 2M Dubai stock — the same band where the AED 750K visa floor was removed. Brokers should expect tenant-to-owner conversions to accelerate as fee waivers and bank pre-approvals stack.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
First-time UAE-resident buyers, brokers pitching renter-to-owner conversions in JVC / Al Furjan / Dubai South / Damac Lagoons, developers without an existing FTHBP slot eyeing onboarding.
</p>

First low-density inland venture: 2.3M sqft, 1,560 units, 70% landscape, 1km central green spine, 5 phases / 13 buildings at 6-7 floors.

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
