<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">MAF invokes force-majeure at Tilal Al Ghaf — ~25% input-cost jump cited; MAF says construction is on schedule</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Tilal Al Ghaf force-majeure notice flips the off-plan buyer-protection switch — and re-prices delay risk across every comparable Dubai SPA.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Semafor reports Majid Al Futtaim has begun sending 'some customers' at Tilal Al Ghaf force-majeure notifications under their SPAs, citing post-Feb-2026 Iran-war supply-chain disruption. MAF publicly disputes the framing, saying construction continues in line with the established project schedule.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Standard Dubai off-plan SPAs entitle buyers to compensation if handover slips 6–12 months — invoking force majeure can pause that clock entirely. Stonehaven estimates Gulf building-material costs rose ~25% since September 2025 (aluminium, bitumen, copper, nickel) with Jebel-Ali-routed fit-out cargo facing delays.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
If MAF's notice holds, every Dubai off-plan buyer targeting a 2026–27 handover must re-read their SPA's force-majeure clause and request written confirmation from their developer. Sellers of partially-paid contracts should expect harder secondary-buyer questions about Oqood status and remaining payment exposure.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Tilal Al Ghaf off-plan buyers, secondary-market investors holding any Dubai 2026–27 off-plan SPA, brokers handling assignments, and lawyers structuring delay-compensation claims.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">World Bank slashes UAE 2026 GDP to 2.4% — down 2.6pp from January's 5% forecast on Hormuz closure</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">The macro pillar under Dubai's 'fundamentals intact' frame just got revised down 2.6 percentage points for 2026.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
World Bank's June 2026 Global Economic Prospects cuts UAE real GDP from 5% to 2.4% for 2026, with GCC-wide growth slashed 3.1pp to 1.3%. UAE rebounds to 4.1% (2027) and 4.2% (2028) as the regional shock is assumed to fade.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The downgrade is directly tied to the Iran-Israel war and Strait of Hormuz closure. The UAE's cut lands softer than the GCC average (-2.6pp vs -3.1pp) because alternative pipelines bypassing Hormuz partially offset the impact — the 2027–28 recovery assumes the conflict eases and trade normalises.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
A 2.6pp GDP cut for the current year is the first hard macro mark against Dubai property's resilient narrative — the kind of number lenders, REIT analysts and family offices retest underwriting models against. The flip side: World Bank still has UAE expanding 4.1–4.2% in 2027–28, framing this as a one-year shock, not a regime change.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Portfolio allocators, REIT and developer credit analysts, mortgage-LTV desks, and macro-aware property advisors revisiting 2026 underwriting assumptions.
</p>
