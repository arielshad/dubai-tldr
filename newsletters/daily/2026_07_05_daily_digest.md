<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai H1 2026: AED 286.44B / 86K deals — second-best half-year ever</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Dubai's H1 2026 books AED 286.44B / 86K deals — second-best half ever, with ready quietly retaking the lead over off-plan for the first time in months.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
W Capital's Jul-3 half-year read of the DLD tape. H1 2026 clears AED 286.44B ($77.88B) across 86,000+ sales — 12.3% below H1 2025's all-time high but still the second-highest first-half on record. Ready-built stock leads with AED 146.7B / 27,200 deals, while off-plan books AED 139.8B / 58,800 deals.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
71,500 residential unit deals, 7,296 building deals, 7,129 land sales. Total real-estate activity — sales + mortgages + gifts — reaches AED 419.94B across 112,850 transactions.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Ready leading off-plan on value is a genuine composition shift — for the last six months, off-plan share ran 55–65%. Secondary-market end-user activity has been the tape's actual engine, making brokerage inventory in established communities the deeper H1 bet.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Fund LPs and family offices tracking the ready-vs-off-plan book; brokers pitching secondary inventory as the H1 growth story; developers pricing H2 launches.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">H1 2026 off-plan offices: AED 13.1B — more than 2019-2025 combined</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Dubai off-plan offices book AED 13.1B / 1,668 units in H1 2026 — more than the previous seven years combined, with Business Bay capturing 52% of the value.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
DLD-sourced H1 2026 figures from Khaleej Times showing Dubai's off-plan office market has been rewired: AED 13.1B across 1,668 units in six months — eclipsing the AED 5.48B / 1,821-unit combined print from 2019-2025. Five developments (Lumena, Lumena Alta, AHS Tower, Shahrukhz, 31 Above) captured 71.7% of value.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Business Bay leads at AED 6.8B / 476 deals (52% of value). Trade Centre Second books AED 1.7B / 76 deals at ~AED 22M/unit; TECOM Site A AED 1.4B / 498 deals at ~AED 2.8M/unit — SME and family-office floor-plate demand.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Off-plan is now the default acquisition path for Grade A offices — waiting for ready inventory means missing pre-completion pricing. The concentration in five buildings means liquidity is high inside those assets but thin elsewhere.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Owner-occupiers underwriting AED 5–50M office buys, family offices evaluating Grade A exposure, commercial brokers pricing off-plan vs. ready.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">AGBI names Emaar's unnamed AED 200B mega-project as Dubai Estate — behind Dubai Hills, on the Gold Line, backed by Dubai Holding.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Chiller-free vs district cooling: Dubai's AED 4–10K/yr rent premium decoded</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Chiller-free = cooling bundled into rent. District cooling = separate monthly bill. In Dubai's summer, the AED-per-year gap is the whole story.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Gulf News explainer by Zainab Husain on Dubai's two dominant cooling setups. Chiller-free means the landlord absorbs the district-cooling bill and folds it into the ask. District cooling means the tenant gets a separate monthly invoice from Empower, Tabreed, Emicool, Nakheel, Emaar DC or Palm DC.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
A Dh70K/yr rent + separate cooling bill vs Dh74K/yr chiller-free comparison shows the AED 4K/yr rent premium roughly equals the bill it skips. Cooling accounts for ~50% of Dubai's electricity demand, so summer bills swing housing cost materially.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
For renters shortlisting two Dh75-90K units, the setup can dominate the AED-per-year decision — the sticker rent is misleading. The Smart Rental Index doesn't normalise for cooling model, so the comparison must be done by hand.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Long-let brokers pricing JVC / Downtown / Business Bay units; end-user renters comparing shortlisted units on full carrying cost; landlords weighing whether to convert a chiller-paid unit to chiller-free.
</p>
