<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">18,200 H1 deliveries hand Dubai tenants real negotiating power — rents -1.1% QoQ</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Cavendish Maxwell quantifies the supply tipping point — 18,200 H1 deliveries hand renters the strongest negotiating window in years.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
The National's Jun-26 read, anchored on Cavendish Maxwell data, lays out the first hard evidence that Dubai's record handover pipeline is biting rents. Apartments down -0.9% / villas down -2.1% over three months to May — quarterly softness on top of an index still +8.9% YoY.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
H1 2026 brought nearly 18,200 residential units to market (+13.1% vs H1 2025). RERA's smart rent index still governs annual increases, but listed asking rents and renewal asks are easing first — one Dubai Hills Estate tenant moved a renewal from AED 195K to AED 175K (-10%).
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
For landlords, the negotiating playbook flips — renewal automation breaks first, then ask discipline. Underwriting models that bake in 5–7% annual rent escalators on Dubai mid-market apartments should be stress-tested at flat-to-negative for FY 2026.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Landlords renewing leases in 2026, build-to-rent operators sizing absorption, brokers advising on rent caps and negotiation, and buyers stress-testing yield-on-cost when bidding on ready apartment stock.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai's AED 275B H1 2026 launch print — the biggest half-year ever, pacing past 2025's full-year book</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Dubai's H1 2026 just printed AED 275B of launches — a half-step shy of 2025's full-year AED 463B run-rate.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
W Capital's Jun-27 survey aggregates Dubai's H1 2026 launch book: 250 DLD-registered Jan-May projects worth AED 75B combined with Emaar's June AED 200B masterplan to reach a record AED 275B+ half-year tally.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The 250 Jan-May projects carry ~59,400 residential units plus 10,800 villas per DLD data. Emaar's June megaproject layers another ~150,000-resident, 4.5M sqm GFA masterplan on top. W Capital chairman Walid Al Zarooni projects the full-year tally to surpass 2025's AED 463B if H2 launches hold pace.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
This is a supply-pipeline signal, not a transaction print: it tells operators that the off-plan absorption test for 2027-2030 just got materially harder. Cap-rate-sensitive ready holders may face fresh comp pressure as deliveries land; off-plan brokers gain inventory but face thinner per-launch demand.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Off-plan brokers tracking inventory; developer-side sales heads scenario-planning H2 launches; ready-market sellers stress-testing 2027-2030 absorption against an expanding pipeline.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Property Finder May read: buyer-seller gap narrows as 63% still expect drops — rental flow recovers to -20%</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Two months into the truce, Property Finder's index says the bid-ask gap is closing — buyers are still in the driver's seat but the lowball window is fading.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Property Finder's May 2026 consumer survey shows the second consecutive monthly decline in buyers expecting price drops — 63%, down from a post-conflict peak above 70%. Advertised asking prices in May ran ~2% below pre-conflict levels (vs ~1% in April), so sellers are giving ground slowly.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
New rental contract volumes recovered to -20% vs pre-conflict in May, up from -32% in March. The mix is shifting: 47% of May rental activity was fresh moves rather than lease renewals (vs 41% in March), meaning house-hunters are stepping into rentals while they wait.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
This is the cleanest mid-cycle read available. For sellers, the trough on asking prices is closer than headlines suggest; for buyers, the 'wait six months' thesis is decaying. Sam McCone (McCone Properties): 'the lowball approach is fading because serious purchasers now recognise they can secure a good deal without unrealistic discounts.'
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Buy-side brokers calibrating offer levels, secondary sellers deciding whether to drop asks or hold, leasing teams sizing the Q3 contract pipeline, market-research teams updating Q2 narrative.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Two of Dubai's biggest masterplan books now sell the visa and the unit in the same room — fewer hand-offs, fewer dropped buyers.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>

The service covers eligibility checks, document prep and the actual application — handled by an accredited visa-services provider, with the developers acting as facilitators. Existing Meraas/Nakheel customers can also use the service.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The 10-year Golden Visa applies to investors holding property worth at least AED 2M. The new in-centre desk is a guided front-end on the DLD Golden Visa portal — co-locating the visa workflow with the sales journey shortens the time from '20% paid' to 'residency stamped.'
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Overseas HNW buyers running Dubai-plus-visa decisions in parallel, brokers handling international leads, family-office buyers structuring portfolios above the AED 2M Golden Visa floor.

Prices won't rise near-term — developers are in a correction phase as overseas buyers have become more discerning post-conflict.
