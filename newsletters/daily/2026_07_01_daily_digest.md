<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai RTA opens AED 34B Metro Gold Line prequalification — 18 stations, 2032 opening</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Dubai's Gold Line moves from map to tender: AED 34B RFP live, 2032 opening, and every JVC/MBR City/Meydan/JGE underwrite now has a real clock.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
The Roads and Transport Authority opened prequalification for the Metro Gold Line — an AED 34B / USD 9.2B programme covering 18 stations across 15 Dubai areas, running from Al Ghubaiba to Jumeirah Golf Estates. It's the concrete procurement step behind the corridor everyone's been repricing on paper.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Bidders must submit prequalification packages by August 17 2026; design contracts and main contracts award in 2027, roughly five years of construction follows, and grand opening is scheduled September 9 2032. Interchanges at Al Ghubaiba (Green Line), Business Bay (Red Line), Meydan and JGE (Etihad Rail).
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Before this, the Gold Line was a repricing thesis on a map — now it's a live procurement with a specific opening date, converting JVC/MBR City/Meydan/JGE premiums from "metro-adjacent" to "metro-scheduled" for underwriting purposes.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Investors underwriting JVC/MBR City/Meydan/JGE resale on a metro premium; brokers pricing plot options along the 18-station corridor; developers timing off-plan launches to the 2027 contract award.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">DFM-listed REIT books 220 ready-to-rent Jebel Ali Village townhouses for AED 894M and stamps a ~AED 75M annual rent print on the city's secondary-residential income narrative.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Dubai Residential REIT (DFM: DUBAIRR), managed by DHAM REIT Management, announced the AED 894M (USD 243M) forward-purchase of 220 townhouses at Jebel Ali Village — 80 three-bed and 140 four-bed units near Sheikh Zayed Road and Ibn Battuta Mall.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The REIT contracts units from the Dubai Holding residential pipeline ahead of stabilisation, then accretes rent when tenants move in. The implied stabilised gross yield on the AED 75M / ~AED 975M combined base sits in the 7–8% gross range; H1 2026 portfolio additions now stand at 276 units.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
The AED 894M forward-purchase ticket sets a public liquidity floor on Jebel Ali Village 3-/4-BR comps, and the AED 75M stabilised-rent figure is the cleanest institutional yield datapoint published this year — useful anchor for any family-office or broker underwriting in the area.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Income-focused buyers underwriting Jebel Ali Village townhouse yields, brokers selling 3-/4-BR in Dubai Holding masterplans, DFM equity holders tracking DUBAIRR, and secondary sellers in Lantana Hills / Dubai Wharf / The Acres anticipating the next REIT bid.
