<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">fäm Properties + DLD: 69.9% of Dubai primary buyers since 2012 still own — 740,219 properties never resold, retention now matches London/NYC</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Three-quarters of Dubai's post-2012 buyers never sold — fäm + DLD say the flip cycle is gone and retention now matches London/NYC.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
fäm Properties worked with Dubai Land Department to analyse 1.1M+ transactions across 16 years. The headline: 740,219 homes bought since 2012 have never traded again — 69.9% of all primary-market buyers still own their unit.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Cohort analysis shows 42% of 2014 buyers, 53% of 2017 buyers, and 61% of 2022 primary buyers still hold — a steepening retention curve driven by Golden Visa, stronger regulations, and maturing family demand.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
If retention is now structurally London/NYC-like, the resale pipeline is thinner than transaction volumes suggest — most owners aren't sellers, so distress supply is mostly motivated/forced rather than discretionary.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Brokers building seller pitches, off-plan operators sizing post-handover supply risk, ready-market underwriters benchmarking liquidity, and analysts updating Dubai vs global hold-period models.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">First time in years that valuations professionals are openly calling Dubai/UAE a buyer's market — villas confirmed printing at-or-below fair value.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
BlackBrick CEO Matthew Bate confirms villa transactions are clearing "at fair market value and in some cases slightly below" in Victory Heights, JGE and Masakin Al Furjan; CBRE Mena says focus on time-in-market not timing; S&amp;P reaffirmed the UAE's AA / A-1+ sovereign rating in the same week.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
When a Big-3 valuations house, an established prime brokerage, and S&amp;P all line up in the same week, the brokerage narrative hardens into something underwriters can quote — and sellers holding out for 2025 peak prints are now the minority view.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Villa buyers in mid-prime estates, sellers re-pricing 2025 assumptions, valuations and underwriting teams, and family-office allocators needing a long-horizon thesis.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai Municipality readies shared-housing permit portal under Law 4/2026: 12-month grace for existing operators, fines up to AED 1M for repeat violators</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Dubai's shared-housing permit portal is about to go live — partitioned-unit operators have a 12-month grace window to legalise or face escalating fines.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Dubai Municipality confirmed the digital permit portal for shared housing under Law No. (4) of 2026 is being finalised. Existing partitioned-unit operators get a 12-month compliance window once it opens; tenants can no longer sublease partitions — only owners or licensed operators can run shared units.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Permits run 1 year (renewable, 2-year option for owners), require renewal 30 days before expiry, and specify max residents and required facilities. Violations carry fines from AED 500 to AED 500,000, doubling for repeat offences within a year to a maximum AED 1M.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Crackdowns have already targeted Al Rigga, Al Barsha, Al Satwa and Bur Dubai. Once the portal opens, unlicensed shared-housing operators face escalating penalties; conversely, licensed operators get a clean legal path to a high-yield niche that previously sat in a grey zone.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Landlords leasing partitioned units, sublease operators, agencies running co-living inventory, and buyers underwriting sub-AED 1M studios in Deira, Bur Dubai and Al Barsha.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Dubai's first AED 500M beach upgrade phase lands in Mamzar/Deira — a public-amenity catalyst that reframes northern coastal demand and short-let yield.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Dubai Municipality reopened Khor Al Mamzar Beach on 05-May-2026 as the first phase of a Dh3B Dubai-wide beach masterplan: 2.75M sqft destination with 3.6km swim shoreline (+128%), 182,000 sqm sandy beachfront (+110%), a region-first floating walkway, and a 300m 24/7 night-beach segment.
