<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai H1 2026: AED 286.44B / 86,000 Deals — Second-Highest Half-Year Ever, Ready Retakes the Lead</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Dubai's H1 2026 books AED 286.44B / 86K deals — second-best half ever, with ready quietly retaking the lead over off-plan for the first time in months.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
W Capital's Jul-3 half-year read of the DLD tape: H1 2026 clears AED 286.44B ($77.88B) across 86,000+ sales — second-highest first-half print on record. Ready-built stock leads with AED 146.7B / 27,200 deals, flipping the composition from the off-plan-dominant six months prior.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Total activity including mortgages and gifts reaches AED 419.94B across 112,850 transactions. The 12.3% H1 YoY dip aligns with the mid-year cooling narrative but stays well above H1 2024 levels — W Capital frames it as resilience, not an inflection.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Ready leading off-plan on value is a genuine composition shift — it means secondary-market, mortgage-led end-user activity has been the tape's actual H1 engine. That reframes the mid-2026 opportunity toward established-community inventory and secondary-market comps as the more reliable underwriting anchor.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Fund LPs and family offices tracking the ready-vs-off-plan split; brokers pitching secondary-market inventory as the H1 growth story; developers pricing H2 launches against a slightly softer headline.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai Off-Plan Office H1 2026: AED 13.1B / 1,668 Units — More Than the Previous 7 Years Combined</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Dubai off-plan offices book AED 13.1B / 1,668 units in H1 2026 — more than 2019–2025 combined, with Business Bay capturing 52% of the value.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Khaleej Times reports DLD-sourced numbers showing Dubai's off-plan office market printed AED 13.1B across 1,668 units in H1 2026 — surpassing the AED 5.48B combined from 2019–2025. Five developments (Lumena, Lumena Alta by Omniyat, AHS Tower, Shahrukhz by Danube, 31 Above by Beyond) captured 71.7% of value.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Business Bay led with AED 6.8B / 476 transactions (52% of value). Trade Centre Second booked AED 1.7B at ~AED 22M per unit; TECOM Site A AED 1.4B / 498 deals at ~AED 2.8M per unit — pointing to SME and family-office floor-plate demand.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Off-plan is now the default acquisition path for Grade A office — waiting for ready inventory means missing pre-completion pricing. Concentration in five developments means liquidity is deep inside those buildings but thin everywhere else; any handover slippage cascades.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Owner-occupiers underwriting AED 5–50M office buys, PE shops and family offices evaluating Grade A exposure, commercial brokers pricing off-plan vs ready.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Chiller-Free vs District Cooling in Dubai — How the AED 4–10K Rent Gap Actually Works</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Chiller-free = cooling bundled into rent. District cooling = separate monthly bill. In Dubai's summer, the AED-per-year gap is the whole story.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Gulf News features writer Zainab Husain explains Dubai's two cooling setups: chiller-free (landlord absorbs the district-cooling bill, folds it into the ask) vs district cooling (tenant receives a separate monthly invoice from Empower, Tabreed, Emicool, Nakheel, Emaar DC or Palm DC).
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
An illustrative Dh70K/yr rent + separate bill vs Dh74K/yr chiller-free shows the AED 4,000/yr premium is roughly the cooling bill it eliminates — across 10 named communities including JVC, Downtown, Business Bay, JBR and JLT, where district cooling is standard.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
For renters comparing two similarly-priced units, the cooling setup can dominate the full-year cost decision — the sticker rent is misleading. The Smart Rental Index doesn't normalise for cooling model, so full-cost comparison must be done by hand.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Long-let brokers pricing JVC / Downtown / Business Bay units; renters shortlisting on full carrying cost; landlords weighing whether absorbing the cooling bill justifies a higher effective ask.
</p>
