<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">ADREC freezes Abu Dhabi rent increases to 0% — Dubai's 5% RERA cap is now the UAE's most landlord-friendly rental regime</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Abu Dhabi caps every rent at 0% — Dubai's 5% RERA cap is suddenly the UAE's most landlord-friendly rental regime.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
On 2026-06-02, ADREC suspended all rent increases across residential, commercial and industrial leases — both renewals and new contracts on previously-rented units must match the prior contract's rate. The 5% annual cap in place since 2016 is overridden until further notice.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The freeze applies emirate-wide and is effective immediately — landlord notices already served become unenforceable. Dubai's regime is untouched: the RERA Rental Increase Calculator still permits stepped increases up to 20% when the existing rent is materially below the index.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
AD landlords facing margin compression have an incentive to redeploy proceeds into Dubai rental stock where the RERA cap still allows rent-roll growth. The longer the freeze holds without Dubai matching, the wider Dubai's effective yield premium becomes.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Dubai landlords reunderwriting yield, AD-to-Dubai capital rotators, leasing brokers building tenant pitches, and REIT analysts pricing UAE rental regimes.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">AGBI maps Dubai's bubble pattern: Marina off-plan +113% / ready -46% YoY, while Greens +18% ready and JVC +10% ready signal genuine end-user demand</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Dubai stopped trading as one market this spring — Josh Corder maps which postcodes are speculative bubble and which are end-user demand.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
AGBI's 1 June 2026 analysis uses community-level pricing to argue Dubai now has at least three distinct sub-markets. Betterhomes CEO Louis Harding calls it Dubai's first genuine two- or three-tier marketplace — where off-plan prices sprint ahead of ready prices, the buyer is a speculator; where ready outpaces off-plan, the buyer is a resident.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Speculative shape (off-plan up, ready down): Marina (+113% / -46%), Business Bay (+80% / -6%), Palm (+12% / -2.5%). End-user shape (ready up, off-plan flat/down): Greens (-9.7% / +18%), JVC (-1% / +10%), Creek Harbour (-20% / +5.7%). Outlier: Dubai South ready -53% YoY.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Off-plan buyers in Marina, Business Bay, Palm and Dubai South face the widest gap to today's resale value — assignment liquidity and post-handover refinance both get harder if that gap doesn't close. Resale buyers in Greens, JVC, Downtown and Creek Harbour are in markets where ready stock is being absorbed.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Off-plan reservation buyers in Marina / Business Bay / Dubai South, assignment-resale brokers, mortgage lenders sizing LTV on ready vs off-plan, and end-users shopping Greens / JVC / Downtown.
</p>
