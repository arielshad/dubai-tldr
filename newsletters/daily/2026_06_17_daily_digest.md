<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Palm Jebel Ali locks first villa handovers for end-2026</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">After two decades of false starts, Palm Jebel Ali finally moves from contract-award headlines to a real handover calendar — first keys before year-end.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Dubai Holding Real Estate publicly confirms that the first Palm Jebel Ali villa handovers will commence by end of 2026, anchored to the 723-villa Fronds K–P programme already under construction. The April 2026 AED 3.5B contract for Fronds A–F extends the pipeline to ~AED 8.5B committed construction total.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
A confirmed first-handover window re-prices Fronds K–P assignments from "off-plan" to "imminent delivery" — and creates the first secondary comps for Palm Jebel Ali villas against Palm Jumeirah / Jumeirah Bay benchmarks. Watch Fronds A–F resale pricing in Q3 2026 as the first delivery dataset lands.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Assignment buyers on Fronds K–P valuing imminent-delivery premium; original-allocation owners deciding to hold or flip pre-keys; cash buyers eyeing Fronds A–F off-plan; brokers building Palm Jebel Ali vs Palm Jumeirah comp decks.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai tenants weaponise the Smart Rental Index for renewal cuts</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Eighteen months after DLD wired the Smart Rental Index into rent-cap math, brokers say tenants are now actively using it to push back on renewal increases.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Three Dubai brokers — Merlin Real Estate, Amaya & Co, Range International — tell Khaleej Times tenants are now pulling the DLD Smart Rental Index (live since January 2025) to challenge renewal asks, with apartment rents easing from a Q1 2026 peak even as villa renewals continue to climb year-on-year.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The index scores buildings on grade, finishes, location, spatial value and amenities; under RERA's renewal cap, an existing rent more than ~21% below benchmark permits a +20% increase. Tenants pull the index reading themselves via Dubai REST and challenge landlords whose proposed uplift exceeds the schedule — renewals now open with the index print, not the agent's market read.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
For the first time the index is acting as a buyer-protection tool tenants understand and use. Apartment landlords renewing into Q3 2026 must model the index reading before quoting; secondary-market yields could compress as renewals reset toward index. Villa landlords still hold pricing power — the split is structural, not cyclical.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Apartment landlords renewing Q3–Q4 2026 leases; secondary-market apartment underwriters; rent-vs-buy spread modellers; brokers building tenant-side rep mandates.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">fäm Properties' Jan-May 2026 villa tape: luxury rent value is leaning up while volume holds — value-mix, not pure demand, drives the +27% YoY new-contract print.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
