<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai removes AED 750K floor on 2-year property-investor visa; sub-AED 750K stock now visa-eligible</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Single biggest structural change in Dubai property since the post-COVID boom — every sub-AED 750K listing is now visa-eligible for sole owners.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
DLD Cube Centre rule changes effective 1 May 2026 abolish the AED 750,000 minimum property valuation for the 2-year investor visa when the applicant is the sole owner. Joint owners now need only AED 400,000 each, so a couple can co-own an AED 800K unit and both qualify.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Any fully-owned, completed unit qualifies regardless of price for sole owners; visa cost is AED 10,545 new / AED 8,215 renewal every two years. Off-plan does not qualify — the unit must be finished and title registered.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
24% of 2026 ready-home deals were under AED 750K — that entire pool is now newly residency-eligible. CBRE's Matthew Green: "it fundamentally removes barriers to investment in Dubai real estate, creating a larger addressable market for developers to target."
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Mid-market end-users, buy-to-let operators with sub-AED 750K stock, and broker teams pitching first-time investors. Communities most impacted: JVC, International City, Arjan, Dubai Production City, Dubailand, and Dubai Silicon Oasis.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai owners still doubling money post-softening — off-plan +3.3% YoY in April but ready market down 16% as ValuStrat flags first contraction since 2020</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Off-plan up, ready down 16%, but COVID-era owners still booking ~100% on exit — Khaleej Times maps Dubai's two-speed softening.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
ValuStrat's first quarterly contraction since 2020 is masking a sharp split: off-plan apartment volume rose 3.3% YoY in April while ready-property volume fell 16%. Brokers including Merlin Real Estate argue the dip is cyclical — long-hold COVID-era buyers are still exiting at ~100% gains.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
YTD through April 2026 Dubai logged 57,300+ residential sales with 74% off-plan; April alone delivered 13,082 deals — a fresh monthly high. Buyer rotation is showing up at the perimeter: Warsan and International City are picking up institutional and Chinese-buyer interest tied to upcoming Metro extensions.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Ready-market sellers face longer days-on-market and harder negotiations even on prime stock — the ask must now be checked against a 5–15% off-peak band. For off-plan investors, primary-market liquidity is still expanding despite the broader cool.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Resale sellers re-pricing into a softer ready market; long-hold buyers comparing off-plan vs ready entry; brokers underwriting Warsan / International City accumulation plays.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">First sizeable single-asset BTR block on Sheikh Zayed Road — 695 units flowing into central-Dubai rental supply from June, with one mandated leasing channel.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
betterhomes and H&H announced an exclusive leasing partnership on City Tower, a 93-storey residential tower on Sheikh Zayed Road. All 695 units — studios through four-bedrooms — will be operated as a long-term leasing (build-to-rent) block managed by H&H, with betterhomes as the exclusive leasing channel.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
City Tower sits two minutes' walk from Emirates Towers Metro (Red Line), positioned between DIFC, Downtown and Jumeirah. The whole tower stays under H&H ownership — units are not for individual sale — and betterhomes funnels all tenant demand. Move-ins start June 2026.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
A 695-unit institutional BTR block coming online inside the SHZ/DIFC/Downtown corridor in June is a meaningful supply event — Marina, JLT, and JVC dominate the new-supply pipeline, but core SHZ/DIFC has very little single-asset rental supply. Existing landlords in SHZ corridor towers should expect renewal pricing pressure once occupancy builds.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
DIFC/Downtown/Jumeirah-corridor tenants comparing 1-bed and 2-bed options; landlords of comparable apartments along SHZ who need to refresh asks before June; brokerages building corporate-relocation tenant pipelines.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Dubai April 2026 sales hit AED 48B across 13,977 deals; ValuStrat flags first quarterly index contraction since 2020</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Dubai's April print is healthy on the top line but ValuStrat just called the first quarterly index contraction in five years.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Dubai's April 2026 print logged 13,977 deals worth AED 48B — up 3.5% MoM in count and 10.7% MoM in value. Average price is still +16.1% YoY at AED 1,840/sqft, but ValuStrat's residential capital-values index printed 229.2, a 3.8% quarterly drop and the first negative quarter since 2020.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Mix is tilting toward end-user and lower-mid bands: sub-AED 1M units took 23.3% of sales and the AED 1–2M band 34.7%, while AED 5M+ was ~12%. Apartments led with 11,377 deals / AED 24.1B; plots jumped 34.7% to AED 6.6B; primary (off-plan) stock drove AED 35.8B vs AED 12.2B in resale.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
For sellers this is the moment to anchor pricing to recent DLD comps, not 2025 peaks. The divergence between firm headline value and the ValuStrat dip is exactly when building-level comps beat the index narrative.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Buyers underwriting Q2 entries, sellers timing listings, brokers benchmarking against April comps, and analysts tracking the cooling cycle.
